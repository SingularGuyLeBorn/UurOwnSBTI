# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [3.0.0] - 2024-04-10

### Added
- 重构为 React + Vite + TypeScript 现代技术栈。
- 42 种人格类型完整定义（30 亚文化型 + 10 AI 型 + 2 特殊型）。
- 7 层特殊计分逻辑：跳过检测、一键乱选、时间分析、矛盾检测、一致性奖励、陷阱题、置信度系统。
- 混合题型支持：单选 / 多选 / 填空。
- Mad Libs 风格槽位随机文案生成器。
- 分享卡片组件（`ShareCard.tsx`）。
- 基于 shadcn/ui 的 50+ 现代化 UI 组件。

### Removed
- 废弃了早期的纯 HTML/JS 单文件原型版本。
